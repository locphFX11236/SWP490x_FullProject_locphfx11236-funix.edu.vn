DEV
1. Lỡ push cả file .env lên git remote. Hãy chạy các câu lệnh này:
- Lệnh reset commit
    git reset --soft HEAD~1
- Xóa file .env ra khỏi repo
    git rm -r --cached .env
- Xóa khỏi các git history (chon 1 trong 2)
    git filter-branch --index-filter "git rm -rf --cached --ignore-unmatch .env" HEAD
    git filter-branch --force --index-filter 'git rm --cached --ignore-unmatch application/.env.example' --prune-empty --tag-name-filter cat -- --all
- Commit lại
    git add .
    git commit -m '...'
- Đẩy lên remote với lệnh force
    git push origin --force --all
